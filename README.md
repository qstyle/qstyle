## 👋 Обо мне

Собираю веб-продукты **целиком** — от интерфейса до инфраструктуры и деплоя. Senior Frontend с уклоном в Fullstack: микрофронтенды на **Module Federation**, **React / Next.js** на фронте, **Node.js / NestJS + MongoDB / Redis** на бэке, **Go** для расчётов, **Docker Swarm** в проде.

- 🚀 Автор и разработчик SaaS-продукта **Checker Pro** — аналитика для селлеров Wildberries (в проде, платные подписки)
- 🤖 Активно применяю **AI** в разработке (Cursor, Claude Code) и в продуктах: **LLM / RAG**, MCP-серверы, промпт-инжиниринг, автоматизация через n8n / OpenClaw

**[Checker Pro](https://checkerpro.ru)** — SaaS-платформа аналитики для селлеров Wildberries *(в проде)*

**[`yandex-wiki-mcp`](https://github.com/qstyle/yandex-wiki-mcp)** — MCP-сервер для Yandex Wiki
> Чтение и запись страниц вики прямо из Claude Code и других MCP-клиентов.

> NestJS + Fastify · Prisma / SQLite · Vite + React · Nx-монорепо. Доска, где агент работает вместе с человеком (идея → разработка).
